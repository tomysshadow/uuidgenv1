uuidgenv1

	This command line tool creates Version 1 UUIDs. For security reasons, new
	UUID generators create Version 4 UUIDs. The UUIDs created by this tool use
	the MAC address of your machine's Ethernet card, and may be traced to the
	computer where they were created.
